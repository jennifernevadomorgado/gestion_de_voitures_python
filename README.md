# Projet Voiture - Python

Ce projet est un exercice en Python utilisant la programmation orientée objet.

Il permet de gérer des voitures à l’aide de classes et d’un menu interactif.

## Fonctionnalités

- Création de voitures
- Affichage des informations
- Gestion via menu
- Utilisation des classes

## Technologies

- Python
- Programmation Orientée Objet (POO)
